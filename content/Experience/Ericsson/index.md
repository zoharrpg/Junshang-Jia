#### Description
* Used React and Antd framework to create a web-based management system to help the company manage all of the information and processes of products
* Maintained component state in the web application by using Redux
* Designed MySQL database and inserted thousands of lines data from Excel documents
* Used Spring Boot framework to create a backend application and leveraged Mybatis framework to connect it to MySQL database
* Implemented CRUD Methods in the backend application and database
* Packaged the Web Application into jar file and deployed it on the Alibaba Cloud
